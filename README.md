# Juego Estilo Touhou Project

## 1. Introducción

Este es un juego de disparos vertical inspirado en la franquicia Touhou Project. Disfruta de un clásico juego de "danmaku" (muro de balas) con patrones visuales impresionantes y jugabilidad emocionante.

## 2. Requisitos previos

Para jugar, necesitas:

- Un ordenador con Windows, Linux o macOS
- Al menos 2 GB de RAM disponible
- Un procesador con al menos 2 núcleos
- Tarjeta gráfica básica (integrada o dedicada)
- Si quieres compilar el juego, necesitarás herramientas adicionales (opcional)

## 3. Instalación

### Opción 1: Jugar sin compilar (si hay un ejecutable)
1. Descarga o navega a la carpeta del juego
2. Busca el archivo ejecutable (.exe en Windows)
3. ¡Haz doble clic y juega!

### Opción 2: Compilar en Windows (para la versión C++)
1. Abre la carpeta `Touhou_Game_C++2026`
2. Haz doble clic en el archivo `build.bat`
3. Espera a que termine la compilación
4. El juego se creará en la carpeta `build/bin`

### Opción 3: Compilar manualmente (para avanzados)
1. Asegúrate de tener instalado CMake y un compilador de C++
2. Sigue las instrucciones en la carpeta del proyecto correspondiente

## 4. Guía de uso

### Cómo acceder al juego
1. Ejecuta el juego desde el archivo ejecutable
2. Verás el menú principal

## 5. Mecánicas del juego

### Objetivo del juego
Sobrevive el mayor tiempo posible esquivando miles de balas enemigas y derrotando a los oponentes.

### Controles básicos
- **W, A, S, D o flechas del teclado**: Mover tu personaje
- **Barra espaciadora**: Disparar
- **ESC**: Pausar el juego o volver al menú principal
- **Flechas arriba/abajo**: Navegar por los menús

### Cómo empezar
1. Abre el juego
2. Usa las flechas arriba/abajo para seleccionar "Jugar" en el menú
3. Presiona la barra espaciadora para confirmar
4. ¡Empieza a jugar! Mueve tu personaje para esquivar las balas y dispara para atacar.

### Características del juego
- Patrones de balas variados (espirales, anillos, estrellas, disparos dirigidos y ondas)
- 60 FPS para una experiencia fluida
- Visuales inspirados en el clásico estilo Touhou
- Zonas de juego separadas para una mejor experiencia
